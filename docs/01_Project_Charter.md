# Enterprise AI-Powered Project Delivery Platform

## Objective
Build an internal full-stack platform to help employees manage projects, tasks, deadlines, blockers, collaboration, reporting, and AI-generated project summaries.

## Business Problem
Project updates are scattered across emails, meetings, spreadsheets, and chat tools. This causes poor visibility, delayed delivery, unclear ownership, and difficulty tracking blockers.

## Proposed Solution
A centralized web application where managers can create projects, assign tasks, track employee progress, monitor deadlines, and generate AI-assisted project summaries.

## Target Users
- Admin
- Manager
- Employee
- Viewer

## Current Status
Phase 1 started. Spring Boot backend is created and User Management API is working.

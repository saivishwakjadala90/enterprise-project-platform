# Architecture

## High-Level Architecture

React Frontend will communicate with the Spring Boot backend using REST APIs.

## Components

### Frontend
- React 18
- TypeScript
- Bootstrap or Material UI

### Backend
- Java 21
- Spring Boot 3.x
- Spring Data JPA
- Spring Security
- REST APIs

### Database
- H2 for local development
- PostgreSQL for production

### Future Enterprise Components
- Kafka for notifications
- MongoDB for comments/activity logs
- Docker for containerization
- AWS for deployment
- Jenkins/GitHub Actions for CI/CD
- AI service for project summaries

## Current Backend Modules
- User Management Module

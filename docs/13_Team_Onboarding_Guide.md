## Development Workflow

### Step 1

Clone Repository

```bash
git clone <repository-url>
```

### Step 2

Run Application

```bash
mvn spring-boot:run
```

### Step 3

Read Documentation

* Product Roadmap
* Daily Task Plan
* API Specification
* Architecture

## Git Workflow

Create feature branch:

```bash
git checkout -b feature/module-name
```

Commit changes:

```bash
git add .
git commit -m "Implemented feature"
```

Push:

```bash
git push origin feature/module-name
```

Create Pull Request.
